
# Power BI Business Analytics Dashboard

## Project Overview
This project is a Power BI dashboard developed to analyze business data and present meaningful insights for decision-making.  
The report highlights key metrics, trends, and comparisons using interactive visuals.

---

## Objectives
- Analyze business performance
- Track Key Performance Indicators (KPIs)
- Identify trends and patterns
- Support data-driven decisions

---

- Power BI Desktop  
- Power Query (ETL â€“ Extract, Transform, Load)  
- DAX (Data Analysis Expressions)  
- Excel / CSV / Database as data sources  

---

## Data Preparation
- Data imported using Power Query
- Data cleaning steps:
  - Removed duplicates
  - Handled missing values
- Data transformed for analysis

---

## Visualizations Used
- KPI Cards
- Bar and Column Charts
- Line Charts for trend analysis
- Tables and Matrix visuals
- Slicers for filtering data

---

## DAX Usage
- Measures created for:
  - Total Sales
  - Total Profit
  - Average values
  - Growth and comparisons
- Used DAX functions for calculations and analysis

---

## Interactivity
- Slicers for dynamic filtering
- Drill-down and drill-through features
- Tooltips for better understanding

---

##  Conclusion
This project demonstrates how Power BI can be used to transform raw data into actionable insights using effective data modeling, DAX calculations, and visual storytelling.
